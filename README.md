# Boy-Coy Website Clone

A front-end replica of the **Boy-Coy** website built using **HTML**, **CSS**, and **JavaScript**. This project recreates the original website's interactive design, smooth scrolling effects, layered animations, and responsive layout to practice modern front-end development and animation techniques.

> **Note:** This project is created **for educational and portfolio purposes only**. All original design assets and branding belong to the Boy-Coy website.

---




## ✨ Features

* Responsive landing page
* Smooth scrolling using **Locomotive Scroll**
* GSAP-powered animations
* Parallax scrolling effects
* Multi-section layout
* Interactive UI inspired by the original website
* Animated illustrations and layered graphics
* Social media icons using Remix Icons

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)
* ScrollTrigger
* Locomotive Scroll
* Remix Icons

---
